# CompanyBase

CompanyBase is a desktop application designed to help students manage and organize their internship applications and communication with potential companies.

The system provides a structured way to track companies, contacts, application status, and notes throughout the internship application process.

---

## Purpose

CompanyBase is built to support applicants during their search for internship opportunities by keeping all relevant information in one place.  
It helps users maintain an overview of applications, follow-ups, and company interactions.

---

## Features

- Manage company information (CRUD)
- Store and manage contact persons per company
- Live search functionality for quick filtering
- Notes (About & Comments sections)
- JTable-based data management
- Export company data to PDF
- SQLite database integration
- ayered architecture (UI / Service / DAO)

---

## 🛠 Tech Stack

- Java (Swing GUI)
- SQLite
- JDBC
- Apache PDFBox
- MVC-inspired structure (DAO / Service / UI separation)
